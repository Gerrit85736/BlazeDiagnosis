# Labor Rules Test Plan

Add automated tests for these scenarios:

1. Approved part includes dependent labor.
2. Declined part removes dependent labor.
3. Two approved parts in same shared group include shared labor only once.
4. Diagnostic labor remains when marked `not_required`.
5. Discounts reduce taxable subtotal correctly.
6. Tax is calculated after discount.
7. Declined optional service is excluded from totals.
8. All monetary results are rounded to two decimals.
